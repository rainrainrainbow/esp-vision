The following firmware is applicable to the M5Stack AtomS3R-M12 development
board based on ESP32-S3-PICO-1-N8R8.

The default configuration enables the ESP32-S3 target, native TinyUSB CDC/MSC,
UART REPL fallback, the ESP-VISION 8 MiB flash layout, and OV3660 DVP camera
support. The board uses one USB Type-C port for both ROM USB Serial/JTAG
download mode and runtime USB OTG device mode.
