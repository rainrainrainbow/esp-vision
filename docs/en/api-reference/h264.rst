h264 -- H.264 Encoding
======================

:link_to_translation:`zh_CN:[中文]`

The ``h264`` module encodes images into H.264 NAL units, for example to record video to storage or to feed the :doc:`rtsp` server.

.. code-block:: python

   import sensor, h264

   enc = h264.H264Encoder(320, 240, fps=15)
   nal = enc.encode(sensor.snapshot())

.. seealso::

   :doc:`../concepts/codec-streaming` explains the encode-then-stream pipeline and how H.264 keyframes interact with RTSP delivery.
